# Portfolio de Folachadé Alex CAPO-CHICHI

Portfolio professionnel moderne développé avec Next.js 14, TypeScript et Tailwind CSS.

## 🚀 Technologies utilisées

- **Next.js 14** - Framework React avec App Router
- **TypeScript** - Typage statique
- **Tailwind CSS** - Framework CSS utilitaire
- **Lucide React** - Bibliothèque d'icônes

## 📋 Sections

- **Hero** - Présentation avec nom, titre et liens sociaux
- **Projets** - Showcase des projets principaux (AREA, My_Torch, SVR 500)
- **Compétences** - Technologies et frameworks maîtrisés
- **Contact** - Formulaire de contact et liens sociaux

## 🎨 Caractéristiques

- ✅ Design minimaliste et moderne
- ✅ Dark Mode par défaut
- ✅ Responsive (mobile, tablette, desktop)
- ✅ Animations et transitions fluides
- ✅ Performance optimisée

## 🛠️ Installation

1. Installer les dépendances :
```bash
npm install
```

2. Lancer le serveur de développement :
```bash
npm run dev
```

3. Ouvrir [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## 📦 Scripts disponibles

- `npm run dev` - Lance le serveur de développement
- `npm run build` - Construit l'application pour la production
- `npm start` - Lance le serveur de production
- `npm run lint` - Vérifie le code avec ESLint

## 🌐 Déploiement

Le site peut être déployé facilement sur [Vercel](https://vercel.com) :

```bash
vercel
```

## 📝 Personnalisation

Pour personnaliser le portfolio :

1. **Informations personnelles** : Modifier les composants dans `/components`
2. **Projets** : Éditer le tableau `projects` dans `/components/Projects.tsx`
3. **Compétences** : Modifier `skillCategories` dans `/components/Skills.tsx`
4. **Contact** : Mettre à jour les liens dans `/components/Contact.tsx` et `/components/Hero.tsx`
